Pasos para escribir código HTML

  1. Abrir Notepad++
  2. Escribir el esqueleto básico
    • HTML
      • HEAD
        • TITTLE
          • META CHARSET
        • BODY
  3. Rellenar todo el BODY, cerrando todas las etiquetas
  4. Comprobar que son correctas las tabulaciones
  5. Guardar el archivo como loquesea.html
  6. Probar la web en Internet ExplorerGoogle Chrome
  7. Si algo no esta bien, volver al código y corregir el error
