Step to Create a Webpage Template

  1. Downloand and install a text editor.
  2. Use the text editor to create a HTML document.
  3. Add all essential elements to create a webpage: DOCTYPE, html, tittle, meta and body .
  4. Add HTML structural elements: header, nav, main and footer.
  5. Add static contec to the page.
  6. Add comments to note the purpose of each structural elements.
  7. Save your changes.
